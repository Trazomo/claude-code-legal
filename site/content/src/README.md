# Source Code

This directory contains the project's application code — automation scripts, API integrations, and internal tools for the legal operations workflow.

## Structure

```
src/
  contracts/     # Contract parsing and analysis tools
  research/      # Legal research automation
  templates/     # Document template engine
  utils/         # Shared utilities (citation formatter, date calculator)
```

All source code follows the conventions defined in `CLAUDE.md` and the rules in `.claude/rules/`.
